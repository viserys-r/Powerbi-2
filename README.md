# Powerbi-2
This powerbi file contains of 4 to 5 products that are sold between 5 different countries, This data consisits of its manufacturing price, sales , profit etc..
This data can be used to find out the units sold in different countries by powerbi dashboards, charts ec..
The file in the repository contains an actual visualisation of various charts displayed according to the sales of  5 different  countries, and sales of each products its profit and sales. I have also included a screen recording of the visualization of the data in powership bu using various charts , buttons, interactive dashboards etc..
